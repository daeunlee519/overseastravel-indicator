# 🏖️ 플레이스여행검색 Dashboard - 빠른 요약

## 📌 프로젝트 개요
**주간 쿼리 지표 분석 대시보드**  
http://localhost:3000

XLSX 파일을 업로드하면 query별로 area_sc, area_cc의 주간 증감을 자동으로 분석하고 시각화해주는 웹 도구를 개발했음

---

## 🔥 핵심 기능 5가지

### 1️⃣ 파일 업로드
- 드래그 앤 드롭 방식으로 XLSX 파일을 편리하게 업로드할 수 있도록 구현했음
- 파일을 업로드하면 자동으로 파싱하고 중복 데이터를 처리하도록 했음

### 2️⃣ 데이터 조회
- 원하는 쿼리를 선택하여 주간 지표를 상세하게 확인할 수 있도록 했음
- 선 그래프, 막대 그래프, 증감율 그래프 중 원하는 형태로 전환할 수 있게 만들었음

### 3️⃣ 대시보드
- area_sc 기준으로 상위 20개 쿼리를 한눈에 볼 수 있도록 표시했음
- 전체 데이터의 기간별 트렌드를 차트로 시각화했음

### 4️⃣ 필터 분석
- 원하는 기간 범위를 선택하여 해당 기간의 데이터만 조회할 수 있게 했음
- SR Area(otB/otL/otr)별로 데이터를 필터링하여 분석할 수 있도록 구현했음

### 5️⃣ 데이터 관리
- 모든 파일 업로드 이력을 추적하여 기록을 남기도록 했음
- 특정 기간의 데이터를 선택하여 삭제할 수 있는 기능을 추가했음

---

## 💻 시작하는 방법

```bash
cd weekly-tracker
npm install
npm start
```

위 명령어를 실행한 후 브라우저에서 http://localhost:3000 으로 접속하면 됨

## 📂 데이터가 저장되는 위치

- **분석된 데이터**: `weekly-tracker/data/weekly_data.json` (243MB 크기로 저장됨)
- **업로드 기록**: `weekly-tracker/data/upload_history.json` 파일에 기록됨
- **원본 엑셀 파일**: `/Users/user/Documents/쿼리별 주간지표/` 폴더에 보관되어 있음

---

## 🎯 프로젝트 구축에 사용한 주요 프롬프트 (9단계)

1. **초기 설정**: "주간지표 트래킹 웹 애플리케이션을 만들어줘" 
   → Express 서버와 기본 구조를 구축했음

2. **파일 업로드**: "XLSX 파일 업로드 기능을 추가해줘"
   → Multer를 사용한 파일 업로드 API를 구현했음

3. **중복 처리**: "area_sc는 최대값만, area_cc는 합산으로 처리해줘"
   → 중복 데이터를 자동으로 정리하는 로직을 만들었음

4. **차트 구현**: "선/막대/증감율 3가지 그래프 타입을 구현해줘"
   → Chart.js로 인터랙티브한 차트를 개발했음

5. **대시보드**: "Top 쿼리와 전체 트렌드를 보여주는 대시보드를 만들어줘"
   → 전체 현황을 한눈에 파악할 수 있는 화면을 구성했음

6. **필터링**: "기간과 SR Area로 필터링하는 기능을 추가해줘"
   → 다양한 조건으로 데이터를 필터링할 수 있게 구현했음

7. **태깅**: "SR Area를 수동으로 입력할 수 있는 기능을 만들어줘"
   → 드롭다운으로 SR Area를 편집하고 저장하는 기능을 추가했음

8. **삭제 기능**: "기간별로 데이터를 삭제할 수 있게 해줘"
   → 특정 기간의 데이터를 삭제하는 API와 UI를 구현했음

9. **UI 개선**: "모던하고 직관적인 디자인으로 개선해줘"
   → 깔끔한 스타일과 사용자 피드백 기능을 추가했음

## 🛠️ 사용한 기술 스택

- **백엔드**: Node.js와 Express를 사용하여 서버를 구축했음
- **프론트엔드**: Vanilla JavaScript와 Chart.js로 UI를 개발했음
- **데이터 저장**: JSON 파일 기반으로 데이터를 저장하고 관리함

## 🎓 핵심 구현 내용

### 데이터 처리 로직
- 동일한 query와 period 조합에 대해 area_sc는 최대값만 저장하도록 했음
- area_cc는 모든 값을 합산하여 저장하도록 구현했음
- 증감율은 이전 기간 대비 퍼센트로 자동 계산하도록 했음

### 시각화 기능
- Chart.js를 활용하여 인터랙티브한 차트를 제공했음
- 차트 타입을 버튼 클릭만으로 전환할 수 있게 만들었음
- 마우스 호버 시 상세 수치를 볼 수 있는 툴팁을 추가했음

### 사용자 경험
- 드래그 앤 드롭으로 파일을 쉽게 업로드할 수 있도록 했음
- 업로드 진행률을 실시간으로 표시하여 사용자가 진행 상황을 알 수 있게 했음
- 에러 발생 시 친절한 메시지를 보여주도록 구현했음
